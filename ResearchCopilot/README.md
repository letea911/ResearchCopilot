# ResearchCopilot

AI-powered research literature assistant.

## Overview

ResearchCopilot helps researchers discover, organize, and analyze academic literature using AI.

## Project Structure

```
ResearchCopilot/
├── README.md
├── LICENSE
├── .gitignore
├── requirements.txt
├── config/         # Configuration files
├── data/           # Data storage
├── docs/           # Documentation
├── scripts/        # Utility scripts
├── src/            # Source code
└── tests/          # Test suite
```

## Getting Started

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Configure settings in `config/`
4. Run the application
